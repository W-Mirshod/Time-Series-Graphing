A web application for visualizing and analyzing time-based data with interactive charts and statistical tools.

## Features

- **Multiple Chart Types**: Line charts, bar charts, area charts, and scatter plots
- **Data Import/Export**: Support for CSV and JSON data formats
- **Statistical Analysis**: Built-in statistical functions and trend analysis
- **Custom Time Ranges**: Flexible date range selection and filtering
- **Interactive Interface**: Zoom, pan, and hover interactions on charts
- **Real-time Updates**: Dynamic chart updates as data changes

## Tech Stack

- **Backend**: FastAPI
- **Frontend**: JavaScript with interactive charting libraries
- **Database**: SQLite for data storage
- **Deployment**: Docker & Docker Compose

## Quick Start

1. **Navigate to the app**:
   ```bash
   cd Time-Series-Graphing
   ```

2. **Start the application**:
   ```bash
   docker compose up -d
   ```

3. **Open your browser**:
   Navigate to `http://localhost:8008`

## Usage

- Upload your time series data in CSV or JSON format
- Select chart types and customize visualization options
- Apply statistical analysis tools
- Export charts and analysis results
- Set custom date ranges for focused analysis
